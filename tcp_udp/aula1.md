# Revisão Redes I

OSI:

- Camada 1 - Física: Cabos, WiFi;
- Camada 2 - Enlace de dados: Switch;
- Camada 3 - Rede: Roteadores;
- Camada 4 - Transporte: TCP/UTP;
- Camada 5 - Sessão;
- Camada 6 - Apresentação;
- Camada 7 - Aplicação;

- Modelo OSI diz o que um protocolo precisa
- TCP/IP veio antes de OSI, porém teve ajustes para funcionar em uma estrutura parecida com a do OSI

TCP/IP:

- Camada 1 - Acesso à rede (Física e Enlace);
- Camada 2 - Rede;
- Camada 3 - Transporte;
- Camada 4 - Aplicação (Sessão, apresentação, aplicação);

PDU:

- Camada 4: Dados;
- Camada 3: Segmento/Datagrama
- Camada 2: Pacote
- Camada 1: Quadro - bit
